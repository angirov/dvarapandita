hU~M/
sngon tshe 'das dus bskal pa dang po la //
'og min gsang ba mchog gi dkyil 'khor du //
bcom ldan dpal chen rdo rje gzhon nu yis //
chos kyi dbyings nas kI li'i sgra bsgrags tshe //
rigs kyi khro bo lcam dral tshogs rnams kyis //
so so'i sku mchog gsal bar bstan nas kyang //
ji ltar zhal bzhes dam las ma g.yel bar //
mchod gtor 'di bzhes bcol ba'i phrin las mdzod //
de 'og bar gyi bskal pa dbus ma la //
mon kha ne ring seng+ge rdzong gsum du //
slob dpon chen po pad+ma 'byung gnas dang //
jo mo mkhar chen ye shes mtsho rgyal gyis //
lhag pa'i lha mchog dkyil 'khor bsgrubs pa'i tshe //
dregs pa sde brgyad brtan ma gcu gnyis kyis //
so so'i gzugs mchog gsal bar bstan nas kyang //
ji ltar zhal bzhes dam las ma g.yel bar //
mchod gtor 'di bzhes bcol ba'i phrin las mdzod //
de 'og bskal pa tha ma'i dus kyi tshe //
rnga yab gling phran zangs mdog dpal ri'i rtser //
o rgyan chen po pad+ma thod phreng dang //
rig 'dzin chos dbang lang+ka'i tshogs bcas kyis //
rdo rje phur pa'i dkyil 'khor zhal phyes tshe //
phur srung bcu gnyis ging dang dam can gyis //
so so'i gzugs mchog gsal bar bstan nas kyang //
ji ltar zhal bzhes dam las ma g.yel bar //
mchod gtor 'di bzhes bcol ba'i phrin las mdzod //
da lta sgrub pa'i rig 'dzin bdag cag gis //
mkha' 'gro dam can 'du ba'i gnas mchog 'dir //
tshe 'dir sangs rgyas sku gsum 'grub pa dang //
phrin las rnam bzhi lhun gyis 'grub pa'i phyir //
bka' srung dam can rgya mtsho'i tshogs rnams kyis //
mchod gtor 'di zhes bcol ba'i phrin las mdzod //
ces pa 'di zhe chen rgyal tshab pad+ma rnam rgyal gsung du gsungs shing dil mkhyen rin po che'i thugs dam phur pod las byung ba'o //
