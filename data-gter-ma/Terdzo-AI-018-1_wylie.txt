rin chen gter gyi mdzod la ched du brjod pa ngo mtshar rna ba'i rgyan zhes bya ba bzhugs so //
rab 'byams zhing kun chos 'khor rgya mtsho rtag gnas pa'i //
rgyal ba kun gyi gsung gi rdo rje gcig nyid du //
ljongs 'dir rdo rje slob dpon mchog gi gar rol pa //
gangs can mgon po mtsho skyes bla mas 'gro 'di srungs /
1
/gsung gi rdo rje rgyal ba gnyis pa des //
'phags bod 'dzam gling kun tu chos kyi sgo //
dus gsum rgyun du bstan pas skye rgu kun //
gdod ma'i grub mtha' chen por dbugs dbyin mdzad /
2
khyad par gdong dmar mun pa'i gling chen 'dir //
rigs gsum rnam 'phrul chos rgyal chen po yi //
thugs bskyed khyung sngon skyes kyis mdun bsus pas //
dam chos khri gdugs khor yug bar du brdal /
3
/sprul pa'i lo pan brgya rtsa'i gtsor gyur pa //
mkhan chen zhi 'tsho pan chen bi ma la /
khyad par o rgyan sangs rgyas gnyis pa yi //
bka' drin srid pa'i mthar yang rdzogs ma gyur /
4
/de tshe mdo sngags spyi dang khyad par du //
'od gsal rdo rje theg pa'i bshad sgrub kyi //
bstan pa 'phags yul las kyang gsal lo zhes //
mnyam med a ti sha sogs bsngags par mdzad /
5
/mdo sgyu sems phyogs sgrub chen sde drug sogs //
rma gnyags gnubs so zur gyi skyes mchog las //
smin grol mkha' mthar brdal ba'i bstan pa'i gzhug
/'gyur med rdo rje'i rtsal 'chang de yis bskyangs /
6
/de dus spel ba'i skabs su ma babs shing //
ma 'ongs skal ldan don du dgongs pa yi //
pad+ma'i thugs kyi gces mdzod mthar thug rnams //
phyogs bzhi dbus lnga'i gter khar phyag rgyas btab /
7
/dus la babs tshe pad+ma'i zhal slob kyi //
rnam par 'phrul pa la las gter 'byin cing //
la las 'dzin skyong spel ba'i phrin las brtson //
kun kyang bstan 'gro skyob pa'i dpal mgon nyid /
8
/sngon chad gter chen gter phran du ma byon //
da dung byams mgon bstan pa ma shar bar //
yul chen gter chen yul phran gter phran res //
gsang sngags ston ces lung mang rnams su 'byung /
9
/'on kyang bod 'gro gsar ngas che ba dang //
rten 'byung dbang gis gsar chos brdol tsam nas /
khungs btsun byin rlabs gzi ldan gter rnying sogs //
phal cher zhi ba'i dbyings su mnal bar gyur /
10
/skyes mchog rig 'dzin rtsal 'chang de rnams kyi //
mtshan tsam thos kyang 'khor ba dong sprugs na //
gter chos re re'i smin grol rgyun gnas kyang //
bstan 'gror phan bde skyed pa smros ci dgos /
11
/deng sang dus kyis phal cher sdig la brtson 'grus spyod //
gal chung chos la byed cing 'dun ma tshe 'di gnyer //
blun rnams skyo ma sngas btsan pha phor gtsang la zhen //
zab chos bshad sgrub ka dag zad sar 'khyol la nye /
12
/'di 'dra'i dus su bstan la sems khral dang //
dam chos 'dzin la skyo ba mi mnga' zhing //
log sgrub bzod pa'i go cha gyon pa gang //
rgyal ba'i sras su rgyal bas lung bstan to /
13
/zab pa'i gsang mdzod mtha' dag bsdu ba ni //
rdo rje chos sogs sa bcu'i dbang phyug gi
/spyod yul yin mod nus pa dang 'tsham par //
gsang ba 'dzin pa'i gsang chen bstan pa'i btsas /
14
/de phyir rdor 'bum chos kyi grags pa nas //
mchog gyur bde chen gling pa'i bar byon pa'i //
mchog chen rnam gsum gling pa nyi shu dang //
dri med sde lnga nus ldan nyi shu gcig
15
/gtsor gyur gsar rnying kun la tshad thub tu //
grags pa'i gter chos rnams kyi smin grol dang //
gdams ngag nyer mkho'i yig sna tshang ba bcas //
rin chen rigs 'dus gter gyi mdzod 'di grub /
16
/o rgyan ches bstan zab mo'i chos rnams la //
gzhung bzhin su dang su yis spyod pa kun //
kun gyis bkur sti phu dud bya ba'i yul //
nga sprul yin ces tshad ma'i lung las 'byung /
17
/pad+ma'i chos la mos kun gu ru yi //
zhal mjal smon lam thebs pa'i las 'phro can //
'brel tshad don ldan yin ces ye shes kyi //
mkha' 'gro'i gtso mo mtsho rgyal zhal nas gdams /
18
khyad par gsang mdzod 'di bskrun rig pa 'dzin //
karma ngag dbang yon tan rgya mtsho'i sde //
rgyal ba'i sras po blo gros mtha' yas pa //
pad+ma gar dbang phrin las 'gro 'dul rtsal /
19
/rigs kun khyab bdag pad+ma nyin byed kyis //
sdom pa gsum ldan mtshan gyis dbang bskur te //
'gyur med rdo rje'i rnam rol zhal gyis bzhes //
skyes chen mchog gzigs gzhan yang dgongs pa mthun /
20
/de mtshungs shel phug gsang ba'i lung byang las //
phan tshun gab sprad bai ro'i skye ba ni :
blo gros ming can de la bdag po sprod :
zhes sogs gter gsar shog ser phal mo cher /
21
/bai ro'i rnam 'phrul 'chad rtsom bka' babs par //
lung bstan gter chen rje yis bla mar bkur //
zab chos chog bsgrig dbang bar rjes su gnang /
kho bos kyang ni de mtshungs gzengs bstod byas /
22
/thun mong rnams kyi mthun snang grub par yang //
bstan bcos rtsom la dbang ba'i mtshan nyid tshang //
lhag par dad ldan re mdun pad 'byung re //
dngos su bzhugs zhes lung nyid mi bslu bas /
23
/rtsod bral dus babs sprul pa'i gter ston che //
mchog gyur bde chen gling pa gam nyid du //
byon nas gang gis sngar phan ma snyed pa'i //
gter rnying mang po'i nye brgyud thos pa dang /
24
/sngon chad gter kha kun la ma grags pa'i //
mdo sgyu sems klong rgyud sde'i skor la sogs //
so zur gnubs rnams slar yang byon 'dra ba'i //
ngo mtshar gzhung gdams mang po'i mdzod sgo 'byed /
25
/gter chen gang gis tshogs mchod rgya cher bcas //
o rgyan dngos la gnang ba zhus par yang //
las 'phro skal ldan smon lam shugs yin pas //
ci 'dod bsdu bar rung ba'i bka' lung thob /
26
/de phyir dbang rab skal ldan rnams kyi ngor //
rin chen gter mdzod 'di ru gang bzhugs pa //
o rgyan dngos la nye brgyud 'ded rung bas //
brgyud tshul sogs kyi brtag dpyad ma dgos mod /
27
/snang tshul du yang brgyud pa yod nges shing //
dam sel bslad med khungs btsun legs bsdus pa //
'di ni rdzogs chen bstan pa'i btsas gsum bzhin //
zab gter bstan pa'i btsas su shes par gyis /
